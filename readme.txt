WooCommerce Booking Bundle Hours

== Description ==

WooCommerce Booking Bundle Hours allows you to buy WooCommerce packages of time services,
 use them from time to time and keep track of hours still available and hours already used.

 more info here [WooCommerce Booking Bundle Hours documentation](https://bookingbundles.com/doc/)

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip `woo-booking-bundle-hours.zip`
2. Upload the `woo-booking-bundle-hours` directory (not its contents, the whole directory) to `/wp-content/plugins/`
3. Activate the plugin through the `Plugins` menu in WordPress
